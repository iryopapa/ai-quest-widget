# ゲーミフィケーション導入提案書

## AI教育カリキュラムの学習継続率を高める「育成ゲーム」の組み込み

---

## 1. 本提案の背景

現在のAI教育カリキュラムサイトでは、動画学習・テキスト学習・小テストの流れが整備されており、経験値やポイントの仕組みも導入済みです。しかし、獲得した経験値やポイントの**活用先がない**ため、学習を続けるモチベーションに十分つながっていない状況です。

ターゲットであるママさんたちは、日々の育児の合間を縫って学習に取り組みます。**「今日もやろう」と思える仕掛け**がなければ、どんなに良い教材でも途中で離脱してしまいます。

本提案では、「たまごっち」のようなキャラクター育成ゲームの仕組みをカリキュラムサイトに組み込むことで、**学習すること自体が楽しくなる体験**を設計します。

---

## 2. ゴール

| 目的 | 具体的な指標 |
|---|---|
| 日常的な学習習慣の形成 | 週5日以上ログインする受講者の割合を高める |
| カリキュラム完走率の向上 | 最終章まで到達する受講者を増やす |
| 学習体験の満足度向上 | 「楽しかった」と感じてもらえるカリキュラムにする |

---

## 3. ターゲット像と設計方針

### ターゲット像

- 小さなお子さんがいるママさん
- AI知識はゼロからのスタート
- まとまった時間は取りにくい（主に21〜23時の寝かしつけ後）
- 1回の学習時間は5〜15分

### 設計方針：「ご褒美だけ、罰はなし」

忙しいママさんにとって、「ログインしないとペナルティ」は逆効果です。本提案のゲーミフィケーションは以下の原則で設計します。

- **学習したら良いことがある**（キャラが育つ、アイテムが買える）
- **学習しなくても悪いことはない**（キャラが弱ったり減点されることはない）
- **忙しい日でも1分で参加できる**（デイリーチャレンジ）
- **自分のペースで進められる**（他人との競争はしない）

---

## 4. 提案内容

カリキュラムサイトに以下の5つのゲーム要素を組み込みます。

---

### 4-1. キャラクター育成システム

学習を進めるとキャラクターが成長・進化していく仕組みです。

#### キャラクターの進化

学習で得た経験値に応じて、キャラクターが5段階に進化します。

| 段階 | キャラクター | 到達レベル目安 | イメージ |
|:---:|---|:---:|---|
| 1 | たまごちゃん | Lv.1〜5 | 生まれたばかり、これからの成長が楽しみ |
| 2 | ひよこちゃん | Lv.6〜10 | 好奇心旺盛、一緒に学び始める |
| 3 | こねこちゃん | Lv.11〜20 | 学ぶことが楽しくなってきた |
| 4 | うさぎちゃん | Lv.21〜35 | AIの知識がどんどん身についてきた |
| 5 | ユニコーン | Lv.36〜 | AI秘書マスターへの道 |

#### 経験値の獲得タイミング

現在サイトにある学習行動がすべて経験値に変わります。

| 学習行動 | 獲得経験値 | 獲得コイン |
|---|:---:|:---:|
| 動画を視聴完了 | +20 XP | +5 コイン |
| テキストを読了 | +10 XP | +3 コイン |
| 小テストに合格 | +30 XP | +15 コイン |
| 小テスト満点合格 | +50 XP | +25 コイン |
| デイリーチャレンジ完了 | +10 XP | +5 コイン |

#### キャラクターの表示

サイトの画面右下にキャラクターが常駐します。学習行動をすると、キャラクターが喜ぶアニメーションと共に「+20 XP!」のような通知が表示されます。キャラクターをタップすると、詳しいステータスや他の機能にアクセスできます。

---

### 4-2. 冒険マップ（学習進捗の可視化）

カリキュラム全体を「冒険の旅」として可視化します。

- 各章がマップ上のステージとして表示される
- クリアしたステージには旗が立ち、キャラクターの現在地がわかる
- 次のステージまでの距離（残りの学習項目数）が見える
- 全体の何%まで進んだかが一目でわかる

**効果**: 「あと2つで次のステージ」「全体の60%まで来た」という進捗の可視化が、完走への強い動機づけになります。ゴールが見えると人は頑張れます。

---

### 4-3. デイリーチャレンジ（毎日の学習習慣づくり）

「今日の1問」として、毎日1問だけのミニクイズを用意します。

- これまで学習した範囲からランダムに1問出題
- 正解でも不正解でも経験値がもらえる（参加すること自体が大事）
- **所要時間は1分以内**
- 正解すると追加ボーナス

**効果**: 「今日は忙しいけど1問だけやろう」→ やったらストリーク維持 → 「せっかくだからもう少し...」という流れが生まれます。学習のハードルを極限まで下げることで、毎日アクセスする習慣を作ります。

---

### 4-4. ストリーク（連続学習ボーナス）

連続で学習した日数に応じてボーナスが増えていく仕組みです。

| 連続日数 | ボーナス | 演出 |
|:---:|---|---|
| 3日連続 | コイン×1.2倍 | 炎マーク点灯 |
| 7日連続 | コイン×1.5倍 | キャラが特別な喜び方 |
| 14日連続 | コイン×2.0倍 + 限定アクセサリー | お祝い演出 |
| 30日連続 | 限定称号「学びの達人ママ」 | 特別な演出 |

**ストリーク維持の条件**: デイリーチャレンジの1問を解くだけでOK（1分で維持可能）

**ストリークが途切れても**: 罰則はありません。また1日目からカウントが始まるだけです。

---

### 4-5. ショップ＆着せ替え（コインの使い道）

学習で貯めたコインでアクセサリーや背景を購入し、キャラクターをカスタマイズできます。

#### アクセサリー例

| アイテム | 価格 | 説明 |
|---|:---:|---|
| リボン | 40 コイン | かわいいリボンをつける |
| めがね | 60 コイン | 知的な雰囲気に |
| 王冠 | 100 コイン | がんばった証 |
| 花かざり | 80 コイン | 春らしい装い |
| 星のステッキ | 120 コイン | キラキラ光る特別アイテム |

#### 背景テーマ例

| テーマ | 価格 | 説明 |
|---|:---:|---|
| お花畑 | 100 コイン | 明るい草原の背景 |
| 星空 | 120 コイン | 夜空に輝く星たち |
| 桜並木 | 150 コイン | 日本の春をイメージ |

**効果**: 「あと30コインで王冠が買える → もう1章やろう」という動機が生まれます。コインに使い道があることで、学習の成果が目に見える「ごほうび」に変わります。

## 5. ユーザー体験の流れ（1日の例）

```
21:00  寝かしつけ完了、スマホでカリキュラムサイトを開く
       → 画面右下にキャラクター（こねこちゃん Lv.14）が表示される

21:01  「今日の1問」の通知が出ている
       → タップして1問解答（正解!）
       → 「+10 XP! 🔥 5日連続!」と表示される
       → ストリーク維持完了（ここまで1分）

21:02  「せっかくだから今日の動画も見よう」
       → 第4章の動画を視聴（5分）
       → 「+20 XP! +5コイン!」キャラが喜ぶ

21:08  動画の後の小テストに挑戦
       → 満点合格!
       → 「+50 XP! +25コイン! おめでとう!」
       → レベルアップ! Lv.14 → Lv.15

21:10  冒険マップを確認
       → 「第4章クリア! 全体の65%達成!」
       → 「あと少しでうさぎちゃんに進化!」

21:12  ショップを覗く
       → 「コインが150貯まった! 桜並木の背景を買おう」
       → キャラの背景が変わって嬉しい

21:15  満足して就寝
       → 明日も1問だけでもやろう、と思える
```

## 6. 技術的な実装方法

### 概要

ゲーム機能を**1つのJavaScriptファイル**としてパッケージ化し、現在のカリキュラムサイトに組み込みます。

### サイトへの組み込み

WordPressサイトのフッター部分に以下のようなコードを1行追加するだけで導入できます。

```
<script src="ai-quest-widget.js"></script>
```

### データの保存

受講者のブラウザ内（LocalStorage）にデータを保存します。サーバーやデータベースの追加は不要です。

| 項目 | 説明 |
|---|---|
| サーバー追加 | 不要 |
| データベース追加 | 不要 |
| 既存サイトへの影響 | なし（独立したウィジェットとして動作） |
| 保存されるデータ | キャラクターの状態、レベル、コイン、購入アイテム、ストリーク等 |

### 注意点

- データはブラウザに保存されるため、別の端末や別のブラウザからアクセスすると引き継がれません
- ブラウザのデータを消去するとリセットされます
- 将来的にサーバー連携が必要になった場合は、拡張可能な設計にしておきます

## 7. 既存機能との関係

| 既存機能 | 対応 |
|---|---|
| 動画視聴 | 視聴完了をトリガーにXP・コインを付与 |
| テキスト学習 | 読了をトリガーにXP・コインを付与 |
| 小テスト | 合格・満点をトリガーにXP・コインを付与 |
| 既存の経験値 | キャラクター育成の経験値として統合 |
| 既存のポイント | ショップで使えるコインとして統合 |
| バッジ/称号（実装済み） | そのまま活用、新しいバッジの追加も可能 |
| 章の進行（満点で次へ） | 冒険マップと連動して可視化 |

## 8. 将来の拡張オプション

今回の提案はDB不要の範囲で設計していますが、将来的に以下の拡張が可能です。

| 拡張オプション | 概要 | 必要なもの |
|---|---|---|
| 学習ボード | 受講者同士の進捗が見える | サーバー + DB |
| チームチャレンジ | グループで学習目標を達成 | サーバー + DB |
| 端末間データ同期 | PCとスマホで同じデータ | サーバー + DB |
| 季節イベント | 期間限定クイズ＆限定アイテム | コンテンツ追加のみ |
| 新キャラクター追加 | 進化ルートの分岐 | コンテンツ追加のみ |

---

## 9. まとめ

本提案は、**「学習するとかわいいキャラクターが育つ」**というシンプルな仕組みで、ママさんたちの学習継続を支援するものです。

### 3つのポイント

1. **毎日1分で参加できる** — デイリーチャレンジとストリークで、忙しい日でも学習習慣を途切れさせない
2. **がんばりが目に見える** — キャラクターの成長、冒険マップの進捗、コインで買えるアイテムが、学習の成果を実感させる
3. **罰がない、ご褒美だけ** — やらなくてもペナルティなし。ポジティブな動機づけだけで継続を促す

### 期待される効果

- 毎日サイトを開く習慣ができる
- 「あと少しでレベルアップ」「あと少しでアイテムが買える」という小さな目標が、次の学習につながる
- カリキュラム全体の完走率が向上する
- 「学ぶのが楽しかった」というポジティブな学習体験が生まれる
